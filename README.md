# todo_app

A TODO app written with [flutter](https://flutter.dev).

![](todo.gif)

## Features
1. Add Todo
2. Mark Todo as completed
3. Delete Todo
4. Remains counter
5. Error will redo the action and show a snackbar

## Database
1. Firebase Realtime Database

## Getting Started

1. Clone this repo
2. `$ flutter get packages`
3. `$ flutter run`
